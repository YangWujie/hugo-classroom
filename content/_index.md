# Hello Hugo!

用如下命令利用`hugo`创建一个名字为`hugo-classroom`的网站：
```bash
$ hugo new site hugo-classroom
```

```goat
hugo-classroom/
├── archetypes
│   └── default.md
├── config.toml
├── content
│   └── _index.md
├── data
├── layouts
│   └── index.html
├── public
├── resources
│   └── _gen
│       ├── assets
│       └── images
├── static
└── themes
```
